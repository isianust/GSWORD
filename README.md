# ⚔️ GSWORD — 軒轅劍・崑崙秘境（單機版）

> 以經典《軒轅劍網路版》為靈感，重現「崑崙秘境」的 2D ARPG 網頁遊戲。  
> 單機版本，支援桌面鍵盤與行動裝置觸控操作。

---

## 📖 專案簡介

本專案以 DOMO 工作室的《軒轅劍網路版》為藍本，重新打造一款 **2D 即時動作 RPG** 的崑崙秘境探索遊戲。  
玩家可從 **9 大職業** 中選擇（武醫藥劍刀陰符刺 + 全新職業「浪子」），在崑崙秘境 5 層關卡中，以即時 ARPG 戰鬥方式擊敗妖魔，逐步探索秘境深處。

### 🎯 核心特色

| 特色 | 說明 |
|------|------|
| 🗡️ 九大職業 | 武、醫、藥、劍、刀、陰、符、刺 + 全新「浪子」（長笛音律） |
| ⚔️ 即時 ARPG | 邊移動邊攻擊，非回合制 |
| 🎵 198 個技能 | 90% 照搬原版 + 10% 全新技能 |
| ❤️💧🌀 三大資源 | HP（紅）、MP（藍）、氣（金）三條資源系統 |
| 🏔️ 崑崙秘境 | 5 層秘境 + 隱藏層，31+ 張關卡地圖 |
| 👾 25+ 怪物 | 含 5 個多階段 BOSS 戰 |
| 📊 完整 RPG | 等級系統（Lv.99）、裝備（5 品質）、背包、任務 |
| 🗄️ 資料驅動 | 全部遊戲數值存於可配置資料庫（JSON/TS） |
| 📱 雙端操控 | 鍵盤（桌面）＋ 虛擬搖桿（行動裝置） |
| 🖥️ 16:9 響應式 | 1280×720 基礎，自適應各螢幕 |
| 🌏 繁體中文 | 全介面繁體中文 |
| 🎵 音效音樂 | 11+ 首 BGM、48+ 個音效 |

## 🏔️ 崑崙秘境

```
🏘️ 崑崙村落（安全區/起點）
  ↓
📜 第一層：試煉之地（Lv.1-10）─── BOSS: 妖狼王
  ↓
🌿 第二層：毒霧峽谷（Lv.8-18）─── BOSS: 蛇妖女
  ↓
🧊 第三層：冰晶洞窟（Lv.16-26）── BOSS: 冰霜巨像
  ↓
🔥 第四層：火焰深淵（Lv.24-35）── BOSS: 炎龍
  ↓
⚡ 第五層：崑崙之巔（Lv.33-45）── BOSS: 崑崙守護者
  ↓
🌀 隱藏層：混沌虛空（通關後）─── 無盡挑戰 / 噩夢模式
```

> 共 31+ 張關卡地圖，每層有獨特地形機制（毒沼、冰面、熔岩、雷擊等）。
> 詳見 [05-MAP-DESIGN.md](docs/05-MAP-DESIGN.md)

## 🛠️ 技術架構

| 項目 | 技術 | 說明 |
|------|------|------|
| 遊戲引擎 | **Phaser 3** | 成熟 2D 框架 |
| 語言 | **TypeScript** | 型別安全 |
| 建構工具 | **Vite** | 快速 HMR，模組化 |
| 地圖風格 | **格子地圖**（64px） | 參考 GVampireSurvivor |
| 美術風格 | **手繪風 / 仿古中國風** | Placeholder 先行，ComfyUI 換圖 |
| 虛擬搖桿 | **自製** | 無外部依賴 |
| 資料儲存 | **IndexedDB** | 本地存檔 |
| 部署 | **GitHub Pages** | 靜態託管 |
