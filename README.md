# ⚔️ GSWORD — 軒轅劍・崑崙秘境（單機版）

> 以經典《軒轅劍網路版》為靈感，重現「崑崙秘境 / 新手秘境」的 2D 網頁遊戲。  
> 單機版本，支援桌面鍵盤與行動裝置觸控操作。

---

## 📖 專案簡介

本專案以 DOMO 工作室的《軒轅劍網路版》為藍本，重新打造一款 **2D 單機版** 的崑崙秘境探索遊戲。  
玩家將扮演持劍修士，運用 **劍術** 與 **藥術** 兩大技能系統，在崑崙秘境中擊敗妖魔，逐步探索秘境深處。

### 🎯 核心特色

| 特色 | 說明 |
|------|------|
| 🗡️ 劍術系統 | 基於軒轅劍網路版的劍系技能，包含近戰攻擊、劍氣遠程、範圍劍招 |
| 💊 藥術系統 | 基於軒轅劍網路版的藥系技能，包含回復、增益、解毒、減益敵方 |
| 🏔️ 崑崙秘境 | 新手秘境關卡地圖，由淺入深的怪物配置 |
| 👾 經典怪物 | 還原軒轅劍網路版的怪物設計與屬性 |
| 📱 雙端操控 | 鍵盤（桌面）＋ 虛擬搖桿與按鍵（行動裝置） |
| 🖥️ 響應式設計 | 自適應各種螢幕尺寸，手機平板皆可遊玩 |
| 🌏 繁體中文 | 全介面繁體中文 |

## 🎮 遊戲設計

### 主角 — 劍藥修士

主角擁有「**劍**」與「**藥**」兩大技能樹：

#### ⚔️ 劍系技能（參考軒轅劍網路版劍術）

| 技能名稱 | 類型 | 說明 |
|----------|------|------|
| 斬擊 | 基礎攻擊 | 近身普通攻擊 |
| 劍氣斬 | 遠程攻擊 | 發出劍氣攻擊前方敵人 |
| 旋風斬 | 範圍攻擊 | 以自身為中心旋轉攻擊周圍敵人 |
| 破甲斬 | 強化攻擊 | 降低敵人防禦並造成傷害 |
| 劍舞 | 連續技 | 連續多段攻擊 |

#### 💊 藥系技能（參考軒轅劍網路版藥術）

| 技能名稱 | 類型 | 說明 |
|----------|------|------|
| 回春術 | 回復 | 恢復自身 HP |
| 解毒術 | 淨化 | 解除中毒狀態 |
| 強身術 | 增益 | 暫時提升攻擊力 |
| 鐵壁術 | 增益 | 暫時提升防禦力 |
| 毒霧術 | 減益 | 對敵人施加持續傷害毒素 |

> ⚠️ 以上技能為初步設計，最終版本可能依遊戲平衡調整。

### 👾 怪物設計（崑崙秘境）

基於軒轅劍網路版新手區域的怪物，初步規劃：

| 怪物名稱 | 等級範圍 | 類型 | 特殊能力 |
|----------|---------|------|---------|
| 小妖狐 | Lv.1-3 | 普通 | 無 |
| 山精 | Lv.2-4 | 普通 | 投石攻擊 |
| 毒蠍 | Lv.3-5 | 普通 | 中毒效果 |
| 石魔偶 | Lv.4-6 | 精英 | 高防禦 |
| 妖狼王 | Lv.5-7 | 首領 | 嚎叫增益同伴 |

> ⚠️ 以上怪物為初步設計，將參照原版資料進一步完善。

## 📱 操控方式

### 🖥️ 鍵盤（桌面）

| 按鍵 | 功能 |
|------|------|
| W / ↑ | 上移 |
| S / ↓ | 下移 |
| A / ← | 左移 |
| D / → | 右移 |
| J / Z | 普通攻擊 |
| K / X | 技能 1 |
| L / C | 技能 2 |
| I | 背包 |
| ESC | 選單 |

### 📱 行動裝置（觸控）

- **左側**：虛擬搖桿（移動）
- **右側**：技能按鈕（攻擊、技能 1、技能 2）
- **頂部**：狀態列（HP、MP）
- **右上**：選單按鈕
