## 📐 Sprite Sheet 規格

### 輸出格式要求

```
格式：PNG（透明底，RGBA）
單格方向：面向右方為預設（程式中翻轉處理左方）
Sprite Sheet 排列：水平排列（左到右為幀序列）
命名規則：{角色/怪物}_{動作}_{方向}.png
例：swordsman_walk_down.png
```

### 建議 Sprite Sheet 整合

每職業輸出為 **1-2 張大圖**，程式中用 frame 切割：

```
方式一（推薦）：每動作一張 Sheet
swordsman_idle.png    → 8 格（2幀 × 4方向），128×256 px
swordsman_walk.png    → 16 格（4幀 × 4方向），256×256 px
swordsman_attack.png  → 16 格（4幀 × 4方向），384×384 px
...

方式二：全部合一張
swordsman_all.png     → 78 格合併，需要 JSON atlas 描述
```

## 🎨 Placeholder 策略

開發階段使用以下 Placeholder：

| 元素 | Placeholder 方式 |
|------|-----------------|
| 角色 | 彩色圓形 + 方向三角形 |
| 怪物 | 紅色圓形（大小依等級） |
| BOSS | 大紅色圓形 + 皇冠符號 |
| 技能特效 | 彩色圓形/矩形動畫 |
| 地圖 | Canvas 格子繪製（同 GVampireSurvivor） |
| UI | HTML/CSS 基本元素 |
| 技能圖示 | 純色方塊 + 文字標籤 |
| 音效 | 簡易合成音/免費素材 |

```typescript
// Placeholder 渲染範例
function drawPlaceholderPlayer(ctx, x, y, direction) {
  // 身體
  ctx.fillStyle = '#4488FF';
  ctx.beginPath();
  ctx.arc(x, y, 16, 0, Math.PI * 2);
  ctx.fill();
  // 方向指示
  ctx.fillStyle = '#FFFFFF';
  // ... 畫三角形指向面朝方向
}
```
