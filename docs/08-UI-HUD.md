## ❤️ 狀態條設計

### HP 條（紅色）

```
位置：頂部左側 / 下方角色面板
顏色：漸層紅 #dc2626 → #ef4444
背景：深灰 rgba(0,0,0,0.6)
邊框：圓角 9px
高度：18px（桌面）/ 14px（行動）
文字：白色粗體 "HP: 850 / 1200"
動畫：平滑過渡 0.2s
特效：
  - HP < 30% 時紅色閃爍
  - 受傷時短暫顯示白色（傷害量）
  - 回復時短暫顯示綠色（回復量）
```

### MP 條（藍色）

```
位置：HP 條下方
顏色：漸層藍 #2563eb → #60a5fa
其他同 HP 條
文字：白色粗體 "MP: 320 / 500"
特效：
  - MP < 20% 時藍色閃爍
  - 使用技能時短暫閃白
```

### 氣條（黃色/金色）

```
位置：MP 條下方
顏色：漸層金 #f59e0b → #fbbf24
其他同 HP 條
文字：白色粗體 "氣: 60 / 100"
特效：
  - 氣值滿時金色發光脈動
  - 氣值增加時短暫閃亮
```

### 等級與經驗條

```
Level 顯示：
  位置：角色頭像旁
  字號：20px 粗體
  顏色：#FFD700（金色）
  文字："Lv.25"

EXP 條：
  位置：等級下方
  顏色：漸層紫 #7c3aed → #a78bfa
  高度：12px
  文字："78% (15600/20000)"

EXP+10% 條（累計經驗指示器）：
  位置：EXP 條下方
  顏色：淺紫 #c4b5fd
  高度：8px
  說明：顯示當前等級已獲得經驗佔總需求的每 10% 標記
  用途：讓玩家了解每 10% 的進度感
  文字："12% → 下個 10%: 還需 1600 EXP"
```

## 📱 行動裝置特別 UI

### 虛擬搖桿

```
位置：螢幕左下角
底座大小：120×120 px
控制球大小：48×48 px
透明度：底座 25%、控制球 50%
觸控範圍：底座周圍 +40px 的區域
```

### 技能按鈕（右下角）

```
      [技2]
 [技3]     [技1]
      [攻擊]     ← 最大的按鈕（64×64）
 [技4]     [技5]
      [技6]

按鈕大小：48×48 px（技能）、64×64 px（主攻擊）
透明度：40%（未按下）→ 80%（按下）
間距：12px
```

### 自適應規則

| 螢幕寬度 | 佈局 |
|---------|------|
| ≥ 1024px | 桌面版完整 UI |
| 768-1023px | 精簡桌面版 |
| < 768px | 行動版（搖桿 + 觸控按鈕） |

| 螢幕方向 | 處理 |
|---------|------|
| 橫式 (Landscape) | 正常遊戲佈局 |
| 直式 (Portrait) | 提示旋轉螢幕 / 壓縮 UI |
