# 📜 06 — 任務系統與 NPC Quest System & NPCs

> 主線任務引導玩家推進關卡，支線任務提供額外獎勵。
> NPC 提供對話、商店、任務等互動。

---

## 🎯 主線任務

### 任務流程總覽

```
新手引導 → 第一層任務 → 第二層任務 → 第三層任務
    → 第四層任務 → 第五層任務 → 最終決戰 → 隱藏層解鎖
```

## 📊 難度系統

| 項目 | 說明 |
|------|------|
| 難度模式 | 無難度選擇，透過怪物等級自然調控 |
| 等級壓制 | 怪物等級高於玩家 5 級以上：受到傷害 +30%，造成傷害 -30% |
| 等級優勢 | 玩家等級高於怪物 5 級以上：經驗值 -50% |
| BOSS 難度 | BOSS HP 高、技能複雜，需要學習機制 |
| 死亡懲罰 | 損失 10% 當前金幣，回到上次存檔點 |
| 重試機制 | BOSS 戰可立即重試（不需重走關卡） |
