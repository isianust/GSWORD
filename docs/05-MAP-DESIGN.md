# 🗺️ 05 — 地圖設計 Map Design

> 格子風地圖（參考 GVampireSurvivor），另加世界大地圖。
> 地圖資料以 JSON 配置存儲，支援動態生成與編輯。

---

## 🎨 地圖風格

### 格子地圖系統（參考 GVampireSurvivor）

```
地圖渲染方式：
- 深色底色（依地圖主題變化）
- 半透明格線（rgba 0.2-0.3 透明度）
- 格子大小：64×64 px
- 每格可放置：地板、牆壁、裝飾物、機關、傳送點
- 攝影機跟隨玩家平滑移動（lerp 0.1）
```

| 區域 | 底色 | 格線色 | 裝飾色調 |
|------|------|--------|---------|

## 🌍 世界大地圖

玩家在世界地圖上選擇要前往的區域，解鎖制前進。

```
                    ☁️ 崑崙之巔
                    ╱
              🔥 火焰深淵
              ╱
        🧊 冰晶洞窟
        ╱
  🌿 毒霧峽谷
  ╱
📜 試煉之地
╱
🏘️ 崑崙村落（起點）
              ╲
               🌀 混沌虛空（隱藏）
```

### 世界地圖互動

- 點擊已解鎖區域 → 傳送進入
- 未解鎖區域顯示鎖頭 + 解鎖條件
- 已通關區域顯示星級評價（☆☆☆）
- 可查看各區域怪物等級範圍
- 支援拖曳/縮放查看

## 🌀 隱藏層：混沌虛空（通關後）

### 模式一：無盡挑戰

```
大小：30×30 格（固定）
模式：波次制，每 10 波一個強化 BOSS
特色：
- 每波結束可選一個隨機增益（三選一）
- 記錄最高波次
- 怪物等級隨波次增長
- 每 10 波 BOSS 為歷代 BOSS 強化版
```

### 模式二：噩夢重溫

```
可重打任何一層的噩夢版本：
- 怪物等級 +15
- 怪物多 30%
- BOSS 新增額外技能
- 掉落品質提升
- 獲得噩夢專屬裝備
```

## 🧩 地形元素

| 元素 | 類型 | 效果 | 出現區域 |
|------|------|------|---------|
| 牆壁 | 障礙 | 不可通過 | 全部 |
| 石柱 | 障礙 | 不可通過，可作掩體 | 全部 |

| 毒沼 | 危害 | 站立每秒 1% HP | 第二層 |

| 冰面 | 地形 | 移動慣性滑行 | 第三層 |
| 薄冰 | 危害 | 站立 3s 碎裂 | 第三層 |
| 熔岩 | 危害 | 站立每秒 3% HP | 第四層 |

| 火牆 | 機關 | 定時出現的火牆 | 第四層 |
| 落石 | 機關 | 定時落下，有警示 | 第一、五層 |

| 強風 | 機關 | 吹動角色位移 | 第五層 |
