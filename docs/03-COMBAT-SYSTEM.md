# ⚔️ 03 — 戰鬥系統 Combat System

> 即時 ARPG 戰鬥，邊移動邊攻擊。
> 三大資源：HP（生命）、MP（魔力）、氣（Chi）。

---

## 🎮 戰鬥模式

| 項目 | 設定 |
|------|------|
| 戰鬥類型 | 即時 ARPG（非回合制） |
| 攻擊方式 | 按鍵攻擊，邊移動邊戰鬥 |
| 碰撞檢測 | 圓形碰撞（角色/怪物/投射物） |
| 攻擊判定 | 近戰（扇形/圓形）、遠程（投射物/射線） |
| 鎖定系統 | 無鎖定，全靠方向瞄準 |

## ❤️💧🌀 三大資源系統

### HP（生命值）— 紅色

| 項目 | 說明 |
|------|------|
| 顏色 | 紅色 |
| 歸零效果 | 死亡 → 回到上次存檔點 |
| 自然恢復 | 戰鬥外每秒恢復 1% |
| 回復來源 | 醫術技能、藥品道具、裝備加成 |
| 上限提升 | 升級 + 裝備 + 增益技能 |

### MP（魔力值）— 藍色

| 項目 | 說明 |
|------|------|
| 顏色 | 藍色 |
| 歸零效果 | 無法使用消耗 MP 的技能 |
| 自然恢復 | 每秒恢復 0.5%（戰鬥中/外相同） |
| 回復來源 | 凝神術、魔力藥品、裝備加成 |
| 上限提升 | 升級 + 裝備 + 增益技能 |

### 氣（Chi）— 黃色

| 項目 | 說明 |
|------|------|
| 顏色 | 黃色/金色 |
| 歸零效果 | 無法使用消耗氣的技能 |
| 累積方式 | 普通攻擊命中 +5 氣、受到攻擊 +3 氣、每秒自然 +1 氣 |
| 最大值 | 基礎 100，隨等級和職業增長 |
| 特殊用途 | 部分強力技能僅消耗氣不消耗 MP |
| 設計理念 | 鼓勵主動戰鬥（打怪累積氣）而非消極躲避 |

## 🔢 傷害計算公式

### 物理傷害

```
基礎傷害 = 技能倍率 × 物攻
防禦減免 = 目標物防 / (目標物防 + 100 + 攻擊者等級 × 5)
最終傷害 = 基礎傷害 × (1 - 防禦減免) × 隨機波動(0.9~1.1)
```

### 法術傷害

```
基礎傷害 = 技能倍率 × 法攻
抗性減免 = 目標法防 / (目標法防 + 100 + 攻擊者等級 × 5)
最終傷害 = 基礎傷害 × (1 - 抗性減免) × 隨機波動(0.9~1.1)
```

### 真實傷害

```
真實傷害 = 固定數值（無視防禦）
用途：中毒 DOT、灼燒 DOT 等
```

### 暴擊判定

```
暴擊判定 = random() < 暴擊率
暴擊傷害 = 最終傷害 × 暴擊倍率（預設 150%，可透過裝備/技能提升）
```

### 閃避判定

```
閃避判定 = random() < 閃避率
閃避成功 → 傷害 = 0，顯示「MISS」
```

### 格擋判定

```
格擋判定 = random() < 格擋率（僅武術師和部分技能提供）
格擋成功 → 傷害減少 50%
```

## 🔥 狀態效果系統

| 減速 | 🐌 | 移速/攻速降低 X% | 取最高值 | 時間結束、疾行術 |
| 麻痺 | ⚡ | 無法行動（短暫） | 不疊加 | 時間結束 |
| 沉默 | 🤐 | 無法使用技能 | 不疊加 | 清心術、時間結束 |
| 恐懼 | 👻 | 無法攻擊，亂跑 | 不疊加 | 時間結束、受到傷害 |
| 混亂 | 🌀 | 隨機移動 | 不疊加 | 時間結束、受到傷害 |
| 昏睡 | 💤 | 無法行動 | 不疊加 | 受到傷害解除 |
| 流血 | 🩸 | 每秒損失 X% HP（不受防禦影響） | 可疊加（最多 5 層） | 時間結束、金創藥 |

| 降防 | 🛡️↓ | 防禦力降低 X% | 取最高值 | 時間結束 |

## ⚔️ 攻擊類型

### 近戰攻擊

```
判定區域：以角色面向為中心的扇形或圓形
扇形：60°-180° 角度，1-2 格距離
圓形：以自身為中心，1-3 格半徑
```

### 遠程攻擊（投射物）

```
投射物類型：
- 直線：直線飛行，碰到敵人消失
- 穿透：直線飛行，穿過敵人繼續前進
- 追蹤：自動追蹤最近敵人
- 扇形：向前方扇形散射
- 全方位：360° 全方向發射

投射物屬性：
- speed: 飛行速度（像素/秒）
- damage: 傷害值
- piercing: 穿透次數
- lifetime: 存在時間（秒）
- hitbox: 碰撞半徑
```

### 範圍攻擊（AOE）

```
範圍類型：
- 圓形 AOE：指定中心 + 半徑
- 扇形 AOE：角色面向 + 角度 + 距離
- 矩形 AOE：指定區域
- 全螢幕：影響可視範圍內所有敵人
- 地面效果：在指定位置持續造成傷害

判定頻率：
- 瞬發 AOE：一次性判定
- 持續 AOE：每 X 秒判定一次（如毒霧、法陣）
```

## 🎯 戰鬥流程

```
每幀（每 16ms / 60 FPS）：

1. 讀取玩家輸入（移動 + 技能按鍵）
2. 更新玩家位置
3. 檢查技能冷卻
4. 如果按下攻擊鍵 && 冷卻完畢 && 資源足夠：
   a. 扣除 MP / 氣
   b. 開始技能動畫
   c. 在動畫命中幀（hit frame）進行傷害判定
   d. 對命中的敵人計算傷害
   e. 應用狀態效果
   f. 顯示飄血數字
   g. 開始冷卻計時
5. 更新所有投射物位置 + 碰撞檢測
6. 更新怪物 AI（見 04-MONSTERS-DATABASE.md）
7. 更新所有狀態效果（Tick DOT、檢查過期）
8. 檢查死亡（HP <= 0）
9. 更新 HUD 顯示
```

## 🏷️ 傷害類型標籤

每個技能/攻擊帶有以下標籤：

| 標籤 | 說明 |
|------|------|
| `physical` | 物理傷害，受物防減免 |
| `magical` | 法術傷害，受法防減免 |
| `true` | 真實傷害，無視防禦 |
| `fire` | 火系（灼燒相關） |
| `ice` | 冰系（冰凍/減速相關） |
| `thunder` | 雷系（麻痺相關） |
| `poison` | 毒系（中毒相關） |
| `dark` | 暗系（陰術相關） |
| `holy` | 聖系（符術部分技能） |
| `sonic` | 音系（浪子專用） |
