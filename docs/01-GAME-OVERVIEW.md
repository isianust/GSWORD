# 📖 01 — 遊戲總覽 Game Overview

> GSWORD — 軒轅劍・崑崙秘境（單機版）

---

## 🎯 遊戲定位

| 項目 | 說明 |
|------|------|
| 類型 | 2D ARPG（即時動作角色扮演） |
| 視角 | 俯視角 Top-Down |
| 平台 | 網頁（HTML5），桌面＋行動裝置 |

## ⚔️ 戰鬥系統

- **即時 ARPG**：邊移動邊攻擊，無回合制
- **三大資源條**：
  - ❤️ HP（生命值）— 紅色
  - 💧 MP（魔力值）— 藍色
  - 🌀 氣（Chi）— 黃色（部分技能消耗氣，透過普攻/受擊累積）
- **傷害計算**：物理/法術/真實傷害三種類型
- **狀態效果**：中毒、灼燒、冰凍、麻痺、流血、沉默等
- **暴擊/閃避/格擋**：基於屬性的機率判定

> 詳細戰鬥設計見 `03-COMBAT-SYSTEM.md`

## 🗺️ 地圖系統

### 格子風地圖

採用類似 GVampireSurvivor 的格子地圖風格：
- **格子大小**：64×64 px
- **深色底色** + 半透明格線
- **攝影機**跟隨玩家平滑移動
- 每張地圖有不同主題色調和地形裝飾

### 世界地圖

另有一張大世界地圖，供玩家選擇前往不同區域：
- 崑崙村落（安全區/起點）
- 崑崙秘境各層（5 層 + 隱藏層）
- 各層之間需解鎖通行

> 詳細地圖設計見 `05-MAP-DESIGN.md`

## 📊 RPG 系統

| 系統 | 說明 |
|------|------|
| 等級系統 | 最高 Lv.99，指數經驗曲線 |
| 技能系統 | 每職業 20+ 技能，技能點分配 |
| 裝備系統 | 武器、頭盔、鎧甲、護腕、鞋子、飾品 ×2 |
| 背包系統 | 30-60 格，可擴充 |
| 道具系統 | 藥品、卷軸、材料、任務道具 |
| 存檔系統 | 本地存檔（IndexedDB），多個存檔欄位 |

## 📱 操控方式

### 🖥️ 鍵盤（桌面）

| 按鍵 | 功能 |
|------|------|
| W / ↑ | 上移 |
| S / ↓ | 下移 |
| A / ← | 左移 |
| D / → | 右移 |
| J / Z | 普通攻擊 |
| K / X | 技能 1 |
| L / C | 技能 2 |
| U | 技能 3 |
| I | 技能 4 |
| O | 技能 5 |
| P | 技能 6 |
| 1-6 | 快捷道具 |
| B | 背包 |
| E | 裝備 |
| Q | 任務日誌 |
| M | 地圖 |
| ESC | 選單 |
| TAB | 角色狀態 |

### 📱 行動裝置（觸控）

- **左側**：虛擬搖桿（移動）
- **右側**：技能按鈕陣列（主攻擊 + 6 個技能快捷鍵）
- **上方**：狀態列（HP/MP/氣/EXP）
- **下方**：快捷道具欄
- **右上**：選單按鈕、小地圖
