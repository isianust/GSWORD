## 🎒 背包系統

| 項目 | 設定 |
|------|------|
| 基礎格數 | 30 格 |
| 最大格數 | 60 格 |
| 擴充方式 | 購買背包擴充（每次 +5 格，費用遞增） |
| 堆疊 | 消耗品/材料可堆疊（各自上限） |
| 裝備 | 不可堆疊，每件佔一格 |
| 排序 | 支援按類型/品質/名稱排序 |
| 丟棄 | 可丟棄任何非任務道具 |
| 出售 | 在 NPC 商店可出售（售價 = 買價 30%） |
